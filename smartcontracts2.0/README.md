# GoFundMe Smart Contracts 2.0

A Hardhat-based smart contract project for GoFundMe.

## Installation

```bash
npm install
# or
yarn install
```

## Compile Contracts

```bash
npm run compile
# or
yarn compile
```

## Deploy Contracts

```bash
npm run deploy
# or
yarn deploy
```

## Run Tests

```bash
npm test
# or
yarn test
```

## Networks

- **hardhat**: Local development network
- **ganache**: Ganache local blockchain
- **localhost**: Local network (requires ganache running on port 8545)

## Configuration

Edit `hardhat.config.js` to modify network settings and solidity compiler version.
